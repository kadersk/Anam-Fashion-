# 🏗️ ANAM FASHION - System Architecture

## Overview

ANAM FASHION is an Enterprise-Grade Tailoring Management System designed for professional boutiques. It integrates Customer Management, Production Workflow, Billing, and Analytics into one seamless platform.

## Tech Stack

**Frontend:** React.js + TypeScript + Tailwind CSS
**Backend:** Node.js + Express.js
**Database:** MongoDB Atlas
**Deployment:** Vercel + Railway

## Architecture Layers

1. **Presentation Layer** (React Frontend on Vercel)
2. **API Layer** (Express Backend on Railway)
3. **Data Layer** (MongoDB on Atlas)

## Key Modules

- Customer Management
- Order Processing
- Measurement Tracking
- Craftsman Management
- Billing & Invoicing
- Reports & Analytics
- Real-time Notifications
